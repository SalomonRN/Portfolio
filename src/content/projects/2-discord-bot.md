description: Proyecto de un bot de Discord, escrito en python, para gestionar múltiples funcionalidades en un servidor de Discord, como reproducir música de Youtube, notificaciones de actividad de amigos, mandar mensajes a otros canales de voz sin la necesidad de entrar a este, y entre más. Posee varios comandos de tipo comando de barras y comandos de mensaje. Para conocer mas de estos lee el README del proyecto.
challanges: Uno de los principales desafíos fue crear un proyecto con una buena estructura y modularidad para facilitar la adición de nuevas funcionalidades en el futuro. Otro desafío fue implementar el texto a voz para reproducir los mensajes de los usuarios en los canales de voz, lo cual requirió timepo ya que a veces esta no funcionaba correctamente, además de cambiar las menciones de los usuarios por sus nombres para una mejor experiencia.
